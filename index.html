Vst-sketches

Processing sketches for a Vetrex vector display

Download .zip Download .tar.gz View on GitHub

NYC Resistor hosted a class on rendering art and games on vector displays like the Vectrex game console and Tektronix oscilloscopes.

These Processing sketches are designed to run on a Vectrex monitor.

To learn more about creating rendering vectors and driving these monitors at v.st.
